# README

Короткая инструкция по воспроизведению обучения и запуску оценки в чистом окружении.

## 1. Подготовка окружения

Рекомендуется Python 3.10+.

Установите зависимости:

```bash
pip install numpy pandas pillow scikit-learn joblib matplotlib opencv-python
```

## 2. Подготовка данных

Ожидается такая логика структуры данных:

- solution (скрипты для обучения и использования модели, артефакты)

- train (папка с данными для тренировки)

- val (папка с данными для валидации)

- split.json (основной файл разбиёния на тренировочные и тестовые данные)

# 3. Обучение

Запуск:

```bash
python train.py
```

Что делает скрипт:

- читает train/val сплиты из `../split.json`
- собирает датасеты для `top` и `bottom`
- обучает две `KNeighborsRegressor` модели в обёртке `MultiOutputRegressor` для x, y для вариантов top и bottom
- считает метрику `MED` на валидации
- сохраняет модели:
  - `top_model.joblib`
  - `bottom_model.joblib`

## 4. Запуск оценки / инференса

После обучения появятся файлы:

- `top_model.joblib`
- `bottom_model.joblib`

Для быстрого теста запустите:

```bash
python predict.py
```

Скрипт загрузит нужную модель, нормализует входные координаты `(x, y)` как `(x / 3200, y / 1800)` и вернёт предсказанные координаты. 

Пример использования из кода:

```python
from predict import predict

result = predict(x=100, y=150, source="top")
print(result)  # [pred_x, pred_y]
```

Поддерживаются только источники:

- `top`
- `bottom`

Пеедача параметров из коммандной строки не предусмотрена.

## 5. Визуальная проверка точек

Для визуализации размеченных точек можно использовать:

```bash
python points_visualization.py
```

Скрипт открывает пример из train/val-части и рисует соответствующие точки на изображениях. 

## 6. Что получится в результате

После полного воспроизведения вы получите:

- обученную модель для `top`
- обученную модель для `bottom`
- значение `MED` на валидации для top-door2, bottom-door2
- возможность запускать предсказание по одной точке через `predict.py` 

## 7. Минимальный сценарий запуска

```bash
pip install numpy pandas pillow scikit-learn joblib matplotlib opencv-python
python train.py
python predict.py
```

## Примечание

В коде зафиксирован размер изображения `3200 x 1800`, он используется для нормализации входных координат и ограничения предсказаний.

В названии файла camera_door2_2025-12-19_09-28-36\coodrs_top.jsonвыявлена опечатка. Файл переименован на coords_top.json.

Некоторые точки разметки не имеют сопоставлений (учтнено при обучении).

Также в json файлах пути к изображениям не соответствуют действительности. Реальные пути формируются автоматически в процессе обучения.
